[Docs](http://karma-runner.github.io/0.12/intro/installation.html)
    
    : Por terminal, acceder al root/test,
    : Ejecutar `npm install`
    : Instalar karma-cli con `npm install -g karma-cli`
    : Ejecutar `karma init`
